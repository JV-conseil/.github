## Setting up Bash as default Shell ğŸš

1. Install Bash from Homebrew with `brew install bash` (the default version provided by Apple is outdated).

2. Add Bash from Homebrew install to `/etc/shells` list ğŸ‘‡

    ```bash
    echo "/opt/homebrew/bin/bash" | sudo tee -a "/etc/shells" >/dev/null
    ```

    Enter `sudo cat /etc/shells` to check your list looks like ğŸ‘‡

    ```bash
    /bin/bash
    /bin/csh
    /bin/dash
    /bin/ksh
    /bin/sh
    /bin/tcsh
    /bin/zsh
    /opt/homebrew/bin/bash
    ```

3. Now change Zsh to Bash from Homebrew install on macOS ğŸ‘‡

    ```bash
    chsh -s "/opt/homebrew/bin/bash"
    ```

    Close the Terminal app and launch it again.

4. Enter `echo "${SHELL}"` to test Bash from Homebrew install is the default Shell.

    Check your Bash version with `bash --version | head -1`.

*see: <https://www.cyberciti.biz/faq/change-default-shell-to-bash-on-macos-catalina>*

## Brewfile ğŸ“œ

How to migrate a Homebrew installation to a new location

1. Running `brew bundle dump` will record an installation to a Brewfile

2. and `brew bundle install` will install from a Brewfile.

3. See `brew bundle --help` for more details.

*see: <https://docs.brew.sh/Tips-N'-Tricks#migrate-a-homebrew-installation-to-a-new-location>*
